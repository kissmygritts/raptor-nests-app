# raptor-nests

This is a prototype of the raptor nests application built for NDOW. 

## Project setup
```
git clone https://github.com/kissmygritts/raptor-nests
cd raptor-nests
git checkout prototype

# install dependancies
npm install

# install json-server
npm install -g json-server

# start json server
json-server db.json

# start app
npm run build
```
